# Save scrollbar position

Save scrollbar position for websites

## Setup

```sh
yarn install
# or
npm install
```

## Development

```sh
yarn watch
# or
npm run watch
```

```sh
yarn build
# or
npm run build
```

## Use extension

- Open chrome://extensions/
- Click `LOAD UNPACKED`
- Select `/dist` folder
